# File Converter

## Folder Structure
```
file-converter/
├── input/
│  └── video.mp4
├── output/
│  └── audio.flac
├── .gitignore
├── convert_mp4_to_flac.js
├── convert_wav_to_flac.js
├── package-lock.json
└── package.json
```
## Setup
- Create input folder
- Save target file in input folder
- Run command
```
npm install
```

## Feature

- Wav to flac
```
npm run wavToflac
```
- Mp4 to flac
```
npm run mp4Toflac
```
